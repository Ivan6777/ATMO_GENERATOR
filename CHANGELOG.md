## Підсумок можливостей за етапами

| Етап | Що додано |
|------|-----------|
| Ядро (main) | Формули OMML→MathML (ECMA-376 §22.1) без «ромбів», всі формати зображень + сніфінг, синхронізація відео при експорті, авто-голос за мовою речення |
| UX-спринти | Автозбереження + .atmo + Ctrl+S, undo-снекбари, прогреси/тости, словник вимови, watchdog звуку, error boundary, exit-анімації, канвас (рамкове виділення, вирівнювання, контекстні меню, замки, перетягування слайдів), демо і тур, waveform, автоперевірка вимог до відео |
| Медіа | Відео з AlternateContent (не картинка), живі медіа в MP4-циклі, аудіо-mime, SVGZ/JXL/JXR + конвертація TIFF/HEIC/ICO у PNG, drag&drop відео на слайд |
| Інтерактиви | H5P-точки на відео для всіх типів + «Цікавий факт» (фото + текст) |
| Анімації | Однотипні (3 обрані типи на всю презентацію), регульована тривалість переходів (0.1–5с/слайд), бігучий рядок-титри |
| Вставка з PowerPoint | Текст — текстом зі стилями і розміром (не картинкою), картинки в оригінальному розмірі, відео файлом/перетягуванням |
| Шрифти | 5 рівнів точності: вбудовані у PPTX → системні → Google Fonts → з системи через дозвіл (Local Font Access) → чесна заміна |
| Адаптивність | Згортання обох бокових панелей + автозгортання на екранах ≤1280px |
| WebCodecs (окрема гілка) | Швидкий експорт MP4: рендер у рази швидше реального часу, offline-мікс аудіо |

**Тестове покриття:** 8 сюїтів, 140+ перевірок (формули, мови/голоси, формати, синхронізація, e2e імпорт PPTX з відео/аудіо/SVGZ, автозбереження, MP4-цикл, exit-анімації, бігучий рядок, тривалість переходів, інтерактиви).
