# Platform Atmo (Studio Pro)

**Studio Pro** — це потужний монолітний React-застосунок, розроблений для імпорту, рендерингу, редагування та відтворення презентацій (сумісних з форматом PPTX) безпосередньо в браузері. Проєкт побудовано з акцентом на інтерактивність та навчання, тому він підтримує не лише стандартні слайди, а й інтерактивні завдання у стилі H5P.

## 🌟 Основні можливості

- **Повноцінний рендеринг слайдів**: Відтворення тексту, фігур, зображень та відео з точним збереженням пропорцій (стандарт 16:9).
- **Парсинг Office форматів (OOXML)**: Вбудовані парсери для обробки XML-вузлів (фігури, заливки, стилі ліній).
- **Математичні формули (OMML to MathML)**: Унікальний механізм перетворення формату Office Math (OMML) безпосередньо у веб-стандарт MathML для ідеального відображення рівнянь.
- **Двигун анімацій**: Підтримка переходів між слайдами (Transitions: fade, push, wipe, zoom) та анімацій появи об'єктів (Entrance Animations: bounce, pop, flyIn, swivel) з власними математичними функціями плавності (easing).
- **Інтерактивні H5P віджети**: Можливість додавати на слайди тестові завдання (Multiple Choice), "Правда/Хибність", заповнення пропусків, флеш-картки та гарячі точки для взаємодії з користувачем.
- **Вбудована система логування**: Детальний журнал подій (`Logger`), який відстежує дії в редакторі та дозволяє експортувати історію змін у `.txt` або `.json`.
- **Автономна бібліотека іконок**: Оптимізована система інлайн-SVG іконок, яка замінює зовнішні залежності типу `lucide-react` для легкості збірки.

## 🏗 Архітектура

Проєкт вирізняється тим, що вся бізнес-логіка, парсери, UI-компоненти та управління станом зосереджені в одному великому файлі — `main.jsx` (понад 20 000 рядків коду). 
Така архітектура дозволяє легко розгортати додаток у середовищах типу `babel-standalone` без складної інфраструктури збирачів (bundlers).

### Основні блоки `main.jsx`:
1. **Базові примітиви та іконки**: SVG компоненти.
2. **Конфігурація**: Константи, палітри тем Office (`DEFAULT_THEME_COLORS`), геометрія фігур (`SHAPE_POLYGONS`).
3. **Парсери**: Конвертери OOXML в HTML5/Canvas/SVG пропси.
4. **Рендеринг**: Функції обчислення кадрів анімацій за часом.
5. **Інтерактивність**: Компонент `InteractiveWidget` для завдань.
6. **Головний компонент `App`**: Управління станом редактора, списками слайдів, панелями інструментів і життєвим циклом.

## 🛠 Технологічний стек

- **React.js**: (Hooks: `useState`, `useReducer`, `useRef`, `useEffect`)
- **Vanilla CSS / Inline Styles**: Для стилізації компонентів.
- **Власні алгоритми**: Для мапінгу кольорів Office (`shade`, `tint`, `lumMod`) та математичного парсингу.
- **HTML5 Canvas / SVG**: Для малювання примітивів і слайдів.

## 🚀 Як запустити

Оскільки вся логіка міститься у файлі `main.jsx`, його можна інтегрувати у будь-який React-проєкт або використовувати через CDN-базовані середовища.

1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/Ivan6777/platform_atmo.git
   cd platform_atmo
   ```
2. Підключіть `main.jsx` до вашого React-оточення (Vite, Create React App, Next.js тощо) і відрендереть головний компонент `App`:
   ```javascript
   import React from 'react';
   import ReactDOM from 'react-dom/client';
   import App from './main.jsx';

   ReactDOM.createRoot(document.getElementById('root')).render(
     <React.StrictMode>
       <App />
     </React.StrictMode>
   );
   ```
