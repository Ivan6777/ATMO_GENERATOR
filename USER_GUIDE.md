# Інструкція користувача — Studio Pro (ATMO_GENERATOR)

## 1. Початок роботи
- **Відкрити презентацію:** перетягніть `.pptx` на стартовий екран або «Вибрати файл». Старий `.ppt` спершу збережіть у PowerPoint як `.pptx`.
- **Продовжити роботу:** перетягніть раніше експортований `.mp4` або файл проєкту `.atmo` — проєкт відновиться разом із медіа.
- **Без файлу:** «почати з порожнього слайду» або «✨ спробувати на демо» (2 готові слайди з формулою та анімаціями).
- При першому вході показується **тур на 4 кроки**. Довідка — кнопка 📄 у топбарі, гарячі клавіші — кнопка **?** (або клавіша `?`).

## 2. Редагування слайдів
- **Текст:** подвійний клік — редагування прямо на канвасі (стилі й формули незмінених рядків зберігаються). Панель праворуч: жирний/курсив/підкреслення, вирівнювання, шрифт, кегль, колір, міжрядковий інтервал.
- **Виділення:** клік — один об'єкт; Shift+клік — додати; тягніть по порожньому місцю — рамка. Для групи в панелі з'являються «Вирівняти» і «Розподілити».
- **Переміщення:** мишею (з напрямними-прилипанням, Alt — вимкнути) або стрілками (Shift = 10px). Tab — наступний об'єкт. Shift при resize — пропорції.
- **ПКМ по об'єкту:** дублювати, шар вище/нижче, закріпити (замок від випадкових зсувів), видалити.
- **Слайди зліва:** перетягування — зміна порядку; значки: ✅ озвучено, ⏳ генерується, 🎬 відео, ⚠ помилка, тривалість.
- **Формули** з PowerPoint конвертуються автоматично (дроби, корені, матриці, індекси — живий MathML).

## 2.1 Вставка з PowerPoint (Ctrl+V)
- **Текст**: скопійований текстовий блок вставляється ТЕКСТОМ (не картинкою) — з кеглем, кольором, жирністю, курсивом, шрифтом і розміром блока як в оригіналі.
- **Картинки**: вставляються в оригінальному розмірі й позиції з буфера.
- **Відео**: браузер не отримує відео з буфера PowerPoint (обмеження платформи) — перетягніть відеофайл прямо на слайд (вставиться в точці кидання) або скопіюйте файл у провіднику і Ctrl+V. Відео всередині .pptx імпортуються автоматично.
- **Шари**: у панелі «Шари» відмітьте галочками 2+ об'єкти — кнопка «Об'єднати (N)» зв'яже їх в один (рухаються/анімуються разом), «Розгрупувати» розбирає назад.

## 3. Анімації
- **Поява:** тип/затримка/тривалість у панелі об'єкта або пікером прямо на об'єкті; блоки на таймлайні перетягуються.
- **Зникнення (exit):** секція «Анімація зникнення» — тип + «Зникнути о (с)». Працює поверх появи.
- **Пресети слайда:** Разом / Каскад / По диктору / **«∅ Без анімацій»** — вимкнути все одним кліком.
- **AI:** «Перегенерувати анімації» — ШІ розставить появи синхронно з текстом диктора.
- **Однотипні (3 анімації):** у вкладці «Анімація» оберіть 3 типи (заголовки / медіа / решта) — кнопка «Анімувати всі» застосує лише їх до всієї презентації каскадом (озвучка не чіпається).
- **Тривалість переходу:** у панелі слайда під селектом переходу — поле «Тривалість переходу» (0.1–5с) для кожного слайда; кнопка «типова» повертає стандарт.
- **Пауза між слайдами:** поле «Пауза між слайдами» у панелі слайда — базова витримка наприкінці КОЖНОГО слайда після озвучки й анімацій (типово **3 с**, діапазон 0–10 с, спільна для всієї презентації). Глядач встигає роздивитись кадр перед переходом; 0 — повернути стару миттєву зміну. Зберігається разом із проєктом.
- **Бігучий рядок (титри):** кнопка «🏃 Додати бігучий рядок» у панелі слайда — стрічка з'являється за N секунд до кінця озвучки (типово 5) і пролітає текст справа наліво; текст, момент появи, кольори і кегль налаштовуються в панелі об'єкта.

## 4. Озвучення (TTS)
- Впишіть текст диктора під слайдом → кнопка озвучки. **Мова визначається автоматично:** українські речення — український голос, англійські — британська англійська.
- **«▶ Зразок»** — почути голос до генерації. **«Вимова»** — словник термінів («OSINT → о-сінт»), наголос підкажіть великою літерою або дефісами; текст на слайдах не змінюється.
- Стилі: нейтральний / радісний / серйозний / спокійний / драматичний / розповідний / **енергійний / дружній / викладач**. Два диктори: маркери `[1]` і `[2]`, пауза — `[ПАУЗА]`.
- Пакетна озвучка всіх слайдів — з прогресом і кнопкою «Зупинити». Якщо звук зник — кнопка 🎧 у топбарі перезапустить аудіо без перезавантаження.

## 5. Відео та аудіо на слайдах
- Відео/аудіо з PPTX переносяться автоматично (MP4, MOV, WebM; MP3, WAV, M4A…) і буферизуються одразу — старт без затримок.
- Поведінка відео в панелі: **loop** (зациклити до кінця слайда) / розтягнути / обрізати; тримінг — на таймлайні.
- Звук вбудованого відео потрапляє в експорт ідеально синхронним із картинкою.

## 6. Інтерактиви (H5P-стиль)
- Вкладка «Інтерактив»: тест, правда/хибність, заповнення пропуску, **«★ Цікавий факт» (фото + текст)**, гаряча точка, акордеон, флеш-картка, кнопка.
- У режимі **«Грати»** кожен інтерактив — пульсуюча точка; клік розкриває вміст, ✕ згортає.
- **Перегляд на змонтованому відео:** експортуйте MP4 → відкрийте його назад у редакторі → «Грати» — точки клікабельні.

## 7. Збереження та експорт
- **Автозбереження** у браузері кожні ~3 с (статус у топбарі) + банер «Відновити сесію». **Ctrl+S** — файл `.atmo`.
- **Прев'ю відео (обкладинка):** блок угорі панелі сцен — завантажте картинку, вона стане **першим кадром** експортованого MP4 (саме його плеєри та LMS показують як мініатюру файлу). Картинка стискається до кадру 16:9, зберігається разом із проєктом (.atmo / MP4 / автозбереження); ✕ на мініатюрі — прибрати, клік — замінити.
- **Субтитри у відео:** галочка «Субтитри у відео» в меню «Завантажити» — текст диктора вшивається внизу кадру репліками, синхронно з озвучкою (розмір: малі/середні/великі). Видно й у режимі «Грати». Службові маркери `[ПАУЗА]`, `[1]`/`[2]` в субтитри не потрапляють.
- **«Завантажити»:** MP4 1920×1080 · H.264 · 30 fps · ~9 Мбіт/с, зі звуком або без; аудіо окремо — MP3/WAV. Чекліст покаже неозвучені слайди.
- Після експорту — звіт відповідності вимогам (фактичний бітрейт, кодек). Для H.264 використовуйте **Chrome або Edge**. Експортований MP4 містить проєкт усередині — його можна доредагувати.

## 7.1 Шрифти презентації (щоб не мінялись)
Точність у 5 рівнів: 1) вбудовані у PPTX шрифти вантажаться з файлу (100% збіг — увімкніть у PowerPoint «Файл → Параметри → Зберегти → Вбудовувати шрифти»); 2) системні шрифти; 3) відсутні тягнуться з Google Fonts (з кирилицею); 4) специфічні/комерційні — тост «Взяти з системи»: один клік і дозвіл — шрифт читається прямо з вашого комп'ютера (Chrome/Edge); 5) якщо ніде немає — чесне повідомлення, який шрифт показано заміною.

## 7.2 Маленькі екрани
Обидві бокові панелі згортаються кнопками «/» до вузької смужки; на екранах ≤1280px вони згортаються автоматично. Вкладки інструментів ховаються повторним кліком.

## 8. Якщо щось пішло не так
- Ctrl+Z — скасувати; видалення має кнопку «Повернути»; збій рендера показує картку відновлення (проєкт не втрачається).
- Великі файли (150МБ+) імпортуються довше — не закривайте вкладку, прогрес на екрані.
- Формати EMF/WMF браузер не декодує — на їх місці буде позначений плейсхолдер.
